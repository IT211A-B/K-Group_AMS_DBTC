#**ATTENDANCE MANAGEMENT SYSTEM**

**Task Description:**

    K-group have been tasked into making a web-based Attendance Management System (AMS). Along with it is a collaboration with 3rd year students to implement the backend of the system as a mobile application.The AMS conveniently and efficiently enables teachers to monitor student attendance as well as manage students schedules. While also allowing administrators (i.e: Department heads) to have a centralized platform to supervise faculty attendance, simultaneously are allowed to oversee both students and teacher performance.

**Task Overview:**
To create a web-based Attendance Management Website that enables school faculty to overview student attendance and schedule. Administrators allowed to overview faculty attendance.
